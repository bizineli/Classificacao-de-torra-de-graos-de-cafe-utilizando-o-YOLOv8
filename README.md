# Classificacao-de-torra-de-graos-de-cafe-utilizando-o-YOLOv8
Esse trabalho teve como objetivo classificar o nível de torra de grãos de café utilizando o YOLOv8. O modelo analisou imagens de grãos de café em diferentes níveis de torra e conseguiu de maneira satisfatória diferenciar e classificar os níveis de torra.
